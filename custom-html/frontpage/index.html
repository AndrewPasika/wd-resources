Welcome to the building management system by KaaIoT!

This building management system (BMS) is an example of what you can do using the Kaa platform.

At KaaIoT, we ourselves use this system to control the air conditioning at our offices. To achieve this, we have developed custom thermostat hardware that uses infra-red light to control our AC units.

Kaa features

The BMS helps us become more environment-friendly and green.
Additionally, this system acts as a great example of how standard Kaa
features aid at implementing practical IoT use cases.

Identity management

Manage and display thermostats and their attributes, organize them in groups by various criteria, and control the access permissions.
Learn more

Communication

Handle secure communication between the thermostats and the Kaa platform, manage device authentication, monitor connectivity status.
Learn more

Configuration management

Manage thermostat configurations by setting temperature setpoint, reporting period, reference barometric pressure, and more.
Learn more

Data collection

Сollect, store, process, and visualize device real-time and historical metrics, such as readings of temperature and humidity level, etc.
Learn more

Command invocation

Remotely switch air conditioning by turning ACs on/off. Control thermostats in real time and receive responses.
Learn more

Software updates

Provide devices with updated software over the air, track device software versions, roll back bad updates.
Learn more
